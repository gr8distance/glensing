What is gargantua?

A package registry for Common Lisp. Packages orbit in gargantua's gravitational field, ready to be pulled into your local environment via area51.

Quick Start

# Install area51
$ curl -fsSL https://gargantua.space/install.sh | sh

# Start a new project
$ area51 init my-project

# Add packages from gargantua
$ area51 install alexandria cl-ppcre dexador
